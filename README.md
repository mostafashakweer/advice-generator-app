## Overview

This project is a simple, interactive advice generator. Users can view a random piece of advice and generate a new one by clicking the dice button. It's fully responsive for mobile and desktop, with smooth hover effects and error handling.

### The Challenge

- View the optimal layout depending on the device's screen size
- See hover states for interactive elements
- Generate a new piece of advice by clicking the dice icon

## Features

- ✅ **Responsive Design**: Works perfectly on mobile and desktop
- ✅ **API Integration**: Fetches random advice from external API with loading states and error handling
- ✅ **Custom Divider**: Built with pure CSS (no images) – includes side lines and pause symbols
- ✅ **Interactive Button**: Dice icon with green glow hover effect
- ✅ **Accessibility**: ARIA labels and semantic HTML

## Built With

- **HTML5** – Structure
- **CSS3** – Styling (Flexbox, Grid, Custom Properties)
- **JavaScript (ES6+)** – API calls, event handling
- **Google Fonts** – Manrope font
- **Advice Slip API** – Random quotes
